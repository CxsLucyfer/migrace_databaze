Úkolem je realizovat funkci WordWrap, která dokáže zformátovat text podle zadaných parametru.

Vstupem funkce jsou tri parametry: ASCIIZ (binární nulou ukoncený) retezec ke zformátování Src, šírka formátu (pocet znaku na rádku) Width a zpusob formátování WrapMethod. Formát nabývá hodnot WRAP_LEFT, WRAP_CENTER, WRAP_RIGHT nebo WRAP_BLOCK. Vstupní retezec Src je oznacen kvalifikátorem const, funkce jej tedy nesmí modifikovat. Vstupní retezec obsahuje slova oddelená bílými znaky (mezi slovy muže být i více než jeden bílý znak) a odstavce oddelené prázdnými rádky (odstavce mohou být oddelené více než jednou prázdnou rádkou).

Výstupem funkce je ukazatel na dynamicky alokovanou pamet, která obsahuje zformátovaný vstupní text. Text je uložen v ASCIIZ formátu (binární nulou ukoncený). Funkce WordWrap alokuje operátorem new [] potrebnou pamet pro navracený retezec, volající (testovací prostredí) zajistí uvolnení pameti voláním delete [] na navracený ukazatel.

Návratová hodnota je NULL pokud je behem formátování nalezeno slovo, které se nevejde na rádku nebo je behem formátování do bloku (WRAP_BLOCK) nalezena rádka, na kterou se vejde pouze jedno slovo.
Pravidla formátování

    * Zformátovaný text obsahuje slova oddelená vždy pouze jednou mezerou a odstavce oddelené vždy pouze jedním prázdným rádkem.
    * Pokud na vstupu není žádné slovo, bude výstupem prázdný retezec (alokovaná pamet, ale obsahuje pouze ukoncující binární nulu).
    * Za posledním slovem na rádku nejsou žádné mezery, pouze odrádkování. Odrádkování je i za posledním rádkem textu.
    * Pri zarovnávání na stred (WRAP_CENTER) je text zleva doplnen mezerami tak, aby byl pokud možno vystreden. Pokud je k disopzici lichý pocet mezer, je vlevo o jednu mezeru méne.
    * Pri zarovnávání do bloku jsou vkládány mezery rovnomerne mezi slova. Pokud mezery nelze rozdelit rovnomerne, pak funkce rozširuje mezery postupne zleva na rádku. Tedy pokud máme 5 slov (mezi kterými jsou 4 mezery) a je potreba mezi ne rozdelit 7 znaku mezera, pak mezery budou mít šírky 2-2-2-1..
    *
    * Pri zarovnávání do bloku se poslední rádka v odstavci neroztahuje (je zarovaná vlevo).

Realizovaná funkce nesmí používat C++ datový typ string, datové typy z STL (vector, list, ...) ani regulární výrazy ze standardní knihovny. Použití techto prostredku povede k chybe pri kompilaci. Na vstupu mohou být dlouhé retezce (napr. milióny znaku). Funkce beží v testovacím prostredí, kde má omezené pametové možnosti (pamet dostacuje na vytvorení bloku pro navracený retezec) a omezenou dobu behu 15s (referencní rešení potrebuje méne než 1.5s).

Do odevzdávaného souboru nepatrí nic jiného, zejména ne funkce main. Pro usnadnení odevzdávání lze využít podmínený preklad - ProgTest definuje promennou preprocesoru __PROGTEST__.
Hlavicka požadované funkce:

const int WRAP_LEFT   = 0;
const int WRAP_CENTER = 1;
const int WRAP_RIGHT  = 2;
const int WRAP_BLOCK  = 3;

char * WordWrap      ( const char * Src, int Width, int WrapMethod );
